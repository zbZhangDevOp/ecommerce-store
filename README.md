## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

prisma db commands
```bash
npm prisma generate
npm prism db push

# to reset the database
npx prisma migrate reset 
```
